Installation
============

Welcome to the installation of Pysammos!

The installation of **pysammos** can be done via pip. 

.. code-block:: bash 

    pip install pysammos

The package requires Python 3.7 or higher. It is recommended to use a virtual environment to avoid conflicts with 
other packages in your system. The required dependencies will be installed automatically from the environment yml file:

.. code-block:: bash

    conda env create -f pysammos_env.yml

For development purposes, you can clone the repository from GitHub and install it in editable mode:

.. code-block:: bash
    
    git clone https://github.com/Claudia-Elijas/pysammos.git
    cd pysammos
    pip install -e .

For more details, please refer to the `Pysammos GitHub repository <https://github.com/Claudia-Elijas/pysammos>`_
